Install Angular CLI

Oktober 08, 2021 by rezaldy266

Oke, Blog kali ini tentang cara install Framework Angular CLI di laptop atau computer kalian.

Bahan-bahan yang dibutuhkan

  • NodeJs
  • NPM (Node Package Manager)
  • Teks Editor

Saya bakal jelasin secara singkat kegunaan dari bahan-bahan yang di atas, NodeJs di perlukan untuk menjalakan kode javascript pada sisi server, dan NPM diperlukan karena bisa berfungsi menjadi command line yang digunakan untuk meng-install modul-modul secara dependency, dan teks editor di perlukan yah buat apalagi, kalau bukan buat mengetik kode kita, teks editor yang gw saranin yaitu visual studio code. Oke langsung aja kita mulai

Pertama kalian tekan tombol windows + R lalu ketik cmd, atau ke search windows dan ketik cmd,lalu klik, setelah diklik halaman command line(cmd) bakal terbuka, kalau kalian bertanya? buat kita buka command line? karena di cmd lah kita bisa install angular 2+

Kalau kalian sudah membuka halaman cmd kalian bisa mengetikan di baris cmd itu 

npm install -g @angular/cli

itu baris perintah untuk men install angular di computer kita, dan untuk angular versi berapa yang ter install itu akan otomatis menggunakan versi terakhir atau terbaru, dan cara ini kita meng install angular secara global, kalian juga bisa men install angular menjadi bersifat local atau berjalan hanya didalam sebuah folder.

untuk mengecek angular kalian ter install atau tidak? dan mengecek versi angular berapa yang ter install kalian di cmd tinggal tulis perintah "ng v" atau "ng -v" itu perintah untuk mengecek versi angular yang ada di computer kalian

ketika selesai meng install kalian harus membuat folder angular atau project angular kalian dengan cara di halaman cmd kalian ketikan 

ng new my-project-name

di angular kata "ng" memiliki arti memberi sebuah perintah, disitu arti perintahnya buatlah project angular , kalian bebas memberi nama apa aja di project kalian, contohnya seperti "ng new projectPertama" disitu saya memberi perintah buatlah project angular dengan nama 'projectPertama', nanti secara otomatis di file manager kalian akan terbuat sebuah project yang bernama 'projectPertama', lalu bagaimana kita menjalankan angular atau running angular?

cara untuk running angular pertama cmd kalian harus masuk ke dalam folder project angular kalian, yaitu dengan cara mengetik perintah "cd"

kalian bisa liat gambar di atas, akan saya jelaskan, pertama project angular kalian berada di drive mana, kalau yang saya berada di drive D maka cmd saya harus masuk dulu ke drive D dengan mengetik "D:" setelah masuk ke drive, kalian harus masuk ke project angular kalian dengan mengetikan "cd nama-project", karena nama project angular saya 'projectPertama' maka saya mengetik seperti yang di gambar, bila cmd kalian sudah masuk ke dalam project angular kalian harus mengetik perintah lagi yaitu 

ng serve

itu perintah untuk menjalakan angular, secara default angular akan memberikan kalian http://localhost:4200/ untuk nama port kalian, tapi kalian juga bisa mencustom nama port kalian dengan mengetik 

ng serve --port 6000

disitu saya memberi perintah run angular di port 6000 karena perintah "--port"

Ini cmd yang menjalakan perintah ng serve, di situ kalian tinggal copy url portnya, disitu saya menggunakan url port "http://localhost:4200/", ketika kalian sudah copy kalian pastekan di browser kalian mau itu chrome, firefox atau apa aja, kalian pastekan dikolom pencarian

Bila muncul seperti itu, berarti kalian berhasil meng install angular CLI dengan versi 11 kalian bisa sebut juga angular versi 11 hahaha, kalau misalkan kalian masih belum paham kalian bisa langsung aja ke website resmi angular yaitu https://angular.io/, dan untuk cara install ya kalian bisa langsung lewat link https://angular.io/guide/setup-local

Oke mungkin cukup sekian blog saya tentang cara install angular CLI, mudah-mudahkan kalian dapat memahaminya, kalau kalian pikir install angular +2 itu ribet, itu sama sekali gak ribet kok, gak sampe 10 menit kok 😁 hehehe, mungkin next blog ya nanti gw bakal bahas bagaimana cara create component,service,pipe, tapi nanti install dulu aja angularnya 😄 hihihi, Baik saya ucapkan terimakasih banyak, see youuu...
